# 🎉 Let's Celebrate — Cafe & Party Hall Website

A luxury-styled hospitality website designed to showcase a cafe and event venue. The site highlights premium dining, celebration services, and venue booking options through a modern and visually rich interface.

## 🌐 Live Website
🔗 https://holy-flamingo.10web.cloud/

## 📌 Project Overview
**Let's Celebrate** is a responsive business website created to represent a cafe and party hall brand online. It provides visitors with information about services, menu offerings, event facilities, and booking options in an elegant and user-friendly layout.

The design focuses on:
- Luxury theme styling
- Smooth navigation
- Visual storytelling
- Clear service presentation

---

## ✨ Features
- Elegant homepage with hero section
- Food menu display
- Event & celebration services section
- Booking / contact form
- Testimonials section
- Responsive design for all devices
- High-quality images and modern UI
